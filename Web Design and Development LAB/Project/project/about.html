Peer to Peer Communication

Description

The Chat App is a web application that allows users to communicate with each other in real-time. Users can send messages to each other and view their message history.

Features

  • Real-time messaging
  • User authentication
  • Message history

SDG Goals Fulfilled

The Chat App contributes to the following Sustainable Development Goals (SDGs):

  • Goal 9: Industry, Innovation, and Infrastructure - By providing a platform for communication, the app contributes to the development of infrastructure and innovation.
  • Goal 11: Sustainable Cities and Communities - The app promotes communication and connectivity within communities, contributing to sustainable urban development.
  • Goal 16: Peace, Justice, and Strong Institutions - By facilitating communication and understanding between individuals, the app promotes peaceful and inclusive societies.

Technologies Used

  • AngularJS for the front end
  • Express.js for the back end
  • MongoDB for the database
  • WebSocket for real-time communication

Future Plans

In the future, we plan to add more features to the Chat App, such as group messaging, file sharing, and voice/video calling.
